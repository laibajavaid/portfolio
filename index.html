Etch-a-Sketch

I designed and developed an interactive pixel art application inspired by the timeless joy of an Etch-a-Sketch. With this application, users can unleash their creativity by selecting any color for both the pen and background, offering a personalized canvas for pixel art creation.

The rainbow feature adds a playful touch, covering each square on the grid with a random color for vibrant and dynamic compositions. For added artistic depth, the application allows users to apply shading and lightening effects to their creations.

The flexibility extends to grid selection, accommodating sizes up to 64 x 64 for intricate and detailed pixel art. Built with HTML, CSS, and Javascript, this application provides a user-friendly platform for pixel art enthusiasts to express their imagination with ease.

Weather App

I developed a user-friendly weather application designed to provide up-to-date weather information for any city worldwide. Not only does tha app delivers accurate forecasts but also enhances user experience through dynamic design adaptations.

Utilizing React hooks and conditions in conjunction with the Modern JavaScript Fetch API from OpenWeatherMap, the app allows users to input a city and receive real-time weather information. What sets this project apart is its responsive design – the background image dynamically changes based on the temperature of the searched location.

Colder temperatures trigger a visually reflective background, creating an immersive experience that aligns with the weather conditions. Conversely, warmer climates prompt a different background, ensuring the application not only provides functional data but also adapts its aesthetic to complement the user's environment. This innovative feature showcases the synergy of functionality and design, adding an extra layer of engagement to the user's interaction with the weather app.

Notes App

I developed a versatile note-taking application that combines functionality with an intuitive user interface, leveraging a range of React features and modern web technologies.Users can seamlessly write and save text for each note, with the application storing data in local storage for easy retrieval.

The dynamic search functionality enables efficient organization, and a character limit feature ensures concise notes. Utilizing React components, props, useState(), useEffect(), and hooks, the project showcases the power of React in handling HTML input with efficiency. The visual appeal is enhanced through Flexbox and Grid CSS styles, offering a responsive and visually pleasing layout.

Additionally, the incorporation of a dark and light mode adds a personalized touch to the user experience, providing flexibility in visual preferences. This project marks my initiation into React JS, showcasing a blend of functional and aesthetic considerations in a user-friendly note-taking application.
